Q1. Give a write-up on the Difference between copy by value and copy by reference.

Copy by value Copy by reference
Whenever an element is assigned to another the value of the element is copied to another. for example
let a =10;
let b=a //10 is copied to b		 Whenever an element is assigned to another the address of assigning element is copied to another. for example
let a =[1,2,3,4];
let b=a //address of a is copied to b
Any changes in the element to which the assignment was done is not reflected in to the element from which the data was assigned. for example
let a =10;
let b=a //10 is copied to b
b=5br console.log(a,b) //a=10 , b=5 		since the actual address is passed any changes in the element to which value was assigned is reflected in the varible from which it was assigned. for example
let a =[1,2,3,4];
let b=a //address of a is copied to b
b[1]=6 console.log(a,b)//[1,6,3,4] [1,6,3,4]
Use to assign simple data types like int float char Used to assign derived data types like array, object, json
